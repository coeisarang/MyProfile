SKILL

Language

객체지향 프로그래밍(OOP) 개념을 완벽히 이해하고 설계할 수 있습니다.

포인터, 메모리 직접 관리, 자료구조 구현 등 저수준 프로그래밍 이해을 이해하여 코딩합니다.

간결하고 효율적인 코드를 작성하며, 다양한 라이브러리와 프레임워크를 활용해 데이터 처리할 수 있습니다.

Frontend

데이터 전달 방식인 GET/POST의 차이를 이해하며 백엔드에게 편리하도록 request method를 정확히 지정하여 작성할 수 있습니다.

다양한 기기에서 일관되고 아름다운 디자인을 구현하며, 유지보수와 확장이 쉬운 효율적인 스타일 코드를 작성할 수 있습니다.

이벤트 처리(클릭, 키보드 입력 등)를 이용하여 동적인 화면을 구현 할 수 있습니다,

Backend

Spring MVC 구조로 RESTful API 설계 및 구현할 수 있습니다.

프로젝트 생성, 빌드, 실행, 디버깅 등 IDE 기능을 숙련 하였습니다.

DataBase

Spring 를 이용하여 DBCoonPool, JDBConnect 방식으로 DB 연동을 할 수 있습니다.

인덱스, 트랜잭션, 제약조건에 대한 이해와 적용할 수 있고 SQL (SELECT, JOIN, GROUP BY 등) 쿼리 작성 능숙 합니다.

Etc

git을 이용하여 팀원들과 프로젝트 형상관리를 할 수 있습니다. git의 기본 명령어를 이해하고 적용합니다.
